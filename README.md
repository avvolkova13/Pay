# PAY

Corporate website for PAYWAYS INTERNATIONAL - FZCO.

## Stack

- Next.js 14 App Router
- React 18
- TypeScript
- Global CSS with component-scoped class naming
- Static export

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3000`.

## Notes

This project is a demo-only corporate website. It does not connect real
payments, does not process card data, and does not provide a real customer
cabinet.
